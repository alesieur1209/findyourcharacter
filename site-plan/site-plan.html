Overview

Purpose

This site will act as a large database and filter for cosplayers looking for characters. A user will be able to check tags based on what they look like (female, blonde, blue eyes, etc.) and then the filter will return a list of characters who resemble the user. The user can then use these results to decide which character they want to cosplay as next.

Audience

Cosplayers of all ages, or people simply looking for costume inspiration. Most cosplayers are typically teens and young adults, but anyone can use this site.

Dynamic elements

Characters will be programmed as a series of objects which can be filtered through and returned. The filter will have tags that can be selected, and then the submit button will call results based on these tags. The search bar will also be functional so users can search up characters by name. Results will display a thumbnail image of a character which can be clicked on to show further details. Lastly, users will be able to sign up with an email to create an account for the website. If I have enough time then I'll add benefits to being logged in such as being able to save search presets and certain favorite characters to your profile.

Content

Filter Page

The filter page will contain a series of tags each in their own category. For example: The category of "Hair Color" will have tags including but not limited to "blonde, brown, black, etc." Categories will include but are not limited to "Hair Color, Hair Length, Eye color, Skin Color, Height, Age, etc." No images will be present on the filter page. At the bottom will be a "Submit" button to run the data through the system.

Results Page

The results page will be a series of character images with the character's name listed below each image. The number of images will vary based on what tags the user checked on the filter page. I won't include every character's image here because there's going to eventually be a lot.
